# `English`
# Description
- A room where you can practice random rooms

# Rule
- If the core is broken, the core cannot be regenerated
- If you break core, you get `Players / 2` token

# How to enter
- Click the TP sign and teleport to RRACTICE_SPAWN

# How to exit
- Reconnect the game
