# `English`
# Description
- ranking system based on several criteria
- NPCs are created in order of ranking up to 1st, 2nd, 3rd on the stage in one corner of the map

# stage update
- per 5 min
# criteria
- token
- having room count
- room clear count
- room challenge count

# reward
pay once a week
> 1. 100 token
> 2. 50 token
> 3. 30 token
